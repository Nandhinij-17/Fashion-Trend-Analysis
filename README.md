# Fashion Trend Analysis using Python

## Overview

This project analyzes fashion product data using Python. It performs data cleaning, statistical analysis, and visualization to understand trends in fashion products.

## Features

- Data preprocessing
- Missing value analysis
- GroupBy operations
- Statistical summaries
- Line Plot
- Bar Chart
- Pie Chart
- Histogram
- Scatter Plot
- Box Plot
- Heatmap
- Pair Plot

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Statsmodels
- Plotly

## Dataset

The dataset contains information about fashion products, including:

- Product Name
- Category
- Gender
- Season
- Price
- Sales Count
- Reviews
- Average Rating
- Material
- Color

## How to Run

1. Install required libraries:

```bash
pip install -r requirements.txt
```

2. Open the notebook:

```
Fashion_Trend_Analysis.ipynb
```

3. Run all cells.
